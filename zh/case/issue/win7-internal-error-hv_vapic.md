## 解决方案

使用下面的一种或者多种方法

1. 升级 CPU的微码
    ```
    apt update && apt install -y intel-microcode
    ```
    随后重启主机
2. 更新Windows7的版本
3. 将虚拟机os类型改成linux
4. 使用OVMF虚拟机
